# Marketing Funnel Dashboard Analysis

## Task
Analyze marketing funnel performance and identify key insights using data visualization.

## Tools Used
- Power BI
- Microsoft Excel

## Dataset
The dataset contains marketing campaign data including:
- Sessions
- Leads
- Qualified Leads
- Customers
- Revenue
- Channel
- Campaign
- Date

## Dashboard Features
The Power BI dashboard includes the following visualizations:
- KPI Cards for Customers, Leads, Sessions, and Revenue
- Conversion Rate Gauge
- Channel Performance Analysis
- Decomposition Tree
- Revenue & Lead Distribution (Tree Map)
- Time-based Performance Trends
- Channel Comparison Table
- Interactive Filters for Date, Channel, and Campaign

## Key Insights
- Users drop off mainly at the Lead to Qualified stage.
- Email campaigns generate the highest quality leads.
- Conversion rates can improve through better lead qualification and stronger email marketing.
- The Lead to Qualified stage needs optimization.

## Files Included
- Power BI Dashboard (.pbix)
- Dataset file
- Dashboard screenshot

## Conclusion
This dashboard helps identify high-performing marketing channels and highlights the stages where users drop off, enabling better optimization of marketing strategies.
